### レビュー用メモ（App Review Notes）
```
## Overview

BakuSoku is an unofficial native client for bakusai.com, a Japanese regional internet forum (similar to Reddit but organized by geographic area). The app fetches publicly accessible HTML content from bakusai.com and presents it through a native mobile interface. There is no backend server — all requests go directly to bakusai.com.

## No Account Required

No account or login is needed to use any feature of this app. Simply launch the app to browse forums. To post a reply, tap the compose button on any thread detail screen.

## Features Beyond the Website (Guideline 4.2)

The app provides significant native functionality not available on the bakusai.com mobile website:

1. Ad-Free Experience — The website is heavily ad-supported. This app removes all ads entirely.
2. Dark Mode — Full dark theme support. The website has no dark mode.
3. Favorites Management — Bookmark boards and threads with new-post badge indicators. Website has no such feature.
4. Browse History — Automatic history tracking with new-post checking. Website has no such feature.
5. NG Word Filter — Users can register keywords to hide matching posts. Website has no such feature.
6. In-Thread Search — Full-text search within a thread with highlight and prev/next navigation. Website has no such feature.
7. Reading Position Restore — Remembers where the user left off in each thread and resumes from that position. Website has no such feature.
8. Anchor Popups — Tapping >>N references shows the referenced post in a popup without scrolling. Website requires manual scrolling.
9. Context Menus — Long-press on threads for quick actions (add to favorites, open in browser, etc.).
10. Haptic Feedback — Tactile response on key interactions.

All of the above are implemented natively and do not exist on the website.

## Privacy & Data Handling

- All user data (favorites, history, NG words, read positions) is stored locally on the device using AsyncStorage and SQLite.
- No data is sent to any server operated by us. There is no backend.
- No advertising SDKs, analytics tools, or crash reporting tools are used.
- When a user posts a reply, the request goes directly to bakusai.com — the same as using the website in a browser.
- Privacy policy: https://bakusoku.pages.dev/#/privacy

## User-Generated Content Moderation (Guideline 1.2)

The app displays posts from bakusai.com's forums, an adult-rated (18+) regional bulletin board. The following moderation tools are available in-app:

- **Age Rating**: The app is rated 18+.
- **EULA Agreement**: Users must agree to the Terms of Service (which explicitly state no tolerance for objectionable content or abusive behavior) before posting any content.
- **NG Word Filter**: Users can register keywords (Settings → NG Words) to automatically hide any posts containing those keywords. This immediately removes matching posts from the feed without scrolling past them.
- **Block User**: Long-pressing any post reveals a "ブロック" (Block) option. This adds the poster's display name to the NG filter, immediately hiding all posts from that user across the app.
- **Report Objectionable Content**: Long-pressing any post reveals a "通報" (Report) option, which opens bakusai.com's dedicated report page in the browser. Reports submitted through bakusai.com are reviewed by bakusai.com's moderation team.
- **Contact Developer**: Users can contact the developer directly via the Support page linked in Settings → Support (https://bakusoku.pages.dev/#/support). A dedicated contact email is also accessible from the support page.
- **Developer Response**: The developer commits to reviewing flagged content reports forwarded to the developer within 24 hours.

Content hosting and primary moderation on bakusai.com is handled by bakusai.com's own moderation team. This app provides additional client-side controls on top of the platform's own moderation system.

## Third-Party Content

This app is not affiliated with or endorsed by bakusai.com or its operator. It accesses publicly available content only. The app name "BakuSoku" (meaning "explosive speed" in Japanese) and subtitle ("for 爆サイ.com") clearly indicate it is an unofficial client.

## Test Instructions

1. Launch the app — the board list screen appears immediately.
2. Tap any region (e.g., 関東) → tap any category → tap any board → see thread list.
3. Tap any thread to view posts. Try pull-to-refresh, tap >>N anchors, use the search icon.
4. Long-press a thread in the list to see the context menu (favorites, open in browser).
5. Check the Favorites tab and History tab in the bottom navigation.

## Test Account

Not required. All features are accessible without registration.
```
