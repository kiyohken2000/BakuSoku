# BakuSoku（爆速）

爆サイ.com 非公式ブラウザアプリ

## Features

- 広告なし・シンプル・快適
- MQTT リアルタイム更新
- ダークモード / ライトモード
- お気に入り・既読管理・NG ワード
- iOS / Android 対応

## Tech Stack

- Expo (React Native)
- React Navigation
- MQTT over WebSocket
- Vite + React (Landing Page)

## Structure

```
apps/
├── mobile/   # Expo app
└── web/      # Landing page (Vite + React)
docs/         # Specs & docs
scripts/      # Analysis scripts
```
